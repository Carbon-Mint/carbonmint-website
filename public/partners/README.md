## Logo Specifications:

- **Format**: PNG or SVG (SVG preferred for better quality)
- **Background**: Transparent
- **Minimum Size**: 200x200 pixels
- **Aspect Ratio**: Maintain original aspect ratio
- **File Size**: Keep under 100KB per logo for optimal loading

- Check their official websites

- Use their LinkedIn company page

## How to Add Logos:

1. Download the logo files from official sources
2. Rename them according to the list above (lowercase, hyphen-separated)
3. Save them in this folder (`public/partners/`)
4. Refresh the website - logos will appear automatically!

## Tips:

- **Right-click** on logos on official websites and "Save image as..."
- Use **PNG with transparent background** for best results
- If logo has white background, use image editing tool to remove it
- **SVG format** is best for scalability and small file size
- Test logos on both light and dark backgrounds

## Temporary Solution:

If you can't find a logo immediately, you can:
1. Use a placeholder image
2. Create a simple text-based logo
3. Contact the organization for official brand assets

## Need Help?

If you're having trouble finding a specific logo:
1. Check the organization's "Media Kit" or "Brand Assets" page
2. Look for "Press" or "Downloads" sections
3. Contact their marketing/communications team
4. Use LinkedIn company pages (often have high-res logos)
